## 🔍 Overview

<a href="#wiFiScanner">WiFi Scanner</a> serves home users troubleshooting slow WiFi, IT administrators managing office networks, network engineers diagnosing wireless environments, and anyone who needs to understand what's happening on their wireless network beyond what macOS's standard network tools show.

The <a href="#wiFiScanner">home user troubleshooting use case</a> addresses the most common home network frustrations — slow WiFi in certain rooms, intermittent connectivity, and unknown devices appearing on the network. WiFi Scanner identifies the signal strength issues, channel conflicts, and device inventory that explain these problems and guide their resolution. The <a href="#wiFiScanner">IT administrator use case</a> uses WiFi Scanner for office network management — auditing connected devices, identifying unauthorized access, verifying wireless coverage across the office, and troubleshooting specific connectivity issues with the detailed network information WiFi Scanner provides.

The <a href="#wiFiScanner">router placement optimization use case</a> uses signal strength mapping to find the optimal router position in a space — walking through rooms with WiFi Scanner open to see how signal strength changes and identifying positions that provide better coverage for the whole space. <a href="#wiFiScanner">Security audit use case</a> — identifying all devices connected to the network, checking for unauthorized connections, and verifying that network security settings match what's configured on the router.</p>

<a href="#wiFiScanner">Vendor identification</a> from MAC address OUI databases identifies the hardware manufacturer of each network device — helping identify what type of device an unknown IP address belongs to by showing whether it's an Apple device, a Samsung phone, a Raspberry Pi, or other recognized hardware. <a href="#wiFiScanner">Export and reporting</a> saves scan results for documentation and comparison — exporting device lists, network scans, and signal data for network documentation, change tracking, and reporting.

<a href="#wiFiScanner">Continuous monitoring mode</a> runs ongoing scans rather than one-time discovery — watching for new devices appearing on the network and tracking signal quality changes over time. <a href="#wiFiScanner">Mac App Store distribution</a> provides WiFi Scanner through Apple's standard distribution channel with security review and straightforward installation.</p>

<details>
<summary>What is the difference between 2.4GHz and 5GHz networks?</summary>

2.4GHz has longer range but fewer non-overlapping channels and more interference. 5GHz has shorter range but more channels and less interference in most environments. WiFi Scanner shows both bands and helps identify which performs better for specific locations.

</details>
